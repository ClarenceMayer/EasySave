Run EasySaveConsole_1.0.7.0_x86_x64_arm.appxbundle to download.
If there is an error, it is because of the missing certificate.
To fix it, execute the following instructions:
• Right click on EasySaveConsole_1.0.3.0_x86_x64_arm.appxbundle >properties
• Go to digital signature tab
• Select your user and click on Details
• Display the certificate
• Install a certificate
• Select local computer (administrator rights are required)
• Select place all certificates in the store then Browse
• Select authorized persons then validate
The certificate is now installed so you can now install the remote console for EasySave
